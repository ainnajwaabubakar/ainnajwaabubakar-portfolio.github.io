Smart Link Laundry System

Smart Link Laundry System project designed to simplify the booking process through codes, ensuring a user-friendly interface. Students can reserve laundry machines effortlessly through a dedicated system. A key feature will include timely notifications, alerting users when their washing cycle is about to complete. By enhancing queue management and promoting efficient time management, this system aims to create a more user-friendly and harmonious laundry experience for our student community.

System Objectives

Firstly, as a developer, I need to analyze current laundry operation in UniSZA Besut Campus before developing this system. Secondly, I need to design an appropriate system that can help to solve identified problems. Lastly, I need to develop prototype of Smart Link Laundry System.

Data Gathering

As a system developer, I need to gather data from current laundry users in order to ensure that my system will meet the user requirements.
A survey is conducted to collect data from several respondents in UniSZA Besut.

Technology Used

AI ChatBot technology is used in this system.
This system will allow users to book washing machine slot via the chatbot and with the integration with IoT-enabled laundry machine, this system will provide real-time updates on machine availability and status.

Project Timeline

As a system developer, I took 14 weeks to develop this system.
Several tasks included when developing this system such
as development team formation, requirements gathering and prototype development.
